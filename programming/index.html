EULER #5

April 20, 2021

The task for euler #5 was to find the lowest common multiple (lcm) of the numbers from 1-20. I started by trying a bruteforce approach, which went through every single possible combination to find the answer. This was very slow because of the sheer number of possibilities. I moved to researching how to find the lcm efficiently. In the final version my program, I used the rule that lcm(1, 2, 3, 4, ...) = lcm(lcm(lcm(lcm(1,2), 3), 4), ...). to find the lcm of two numbers, I found the greatest common denominator (gcd), then used the x * y = lcm(x, y) * gcd(x, y) rule to solve for the lcm.
